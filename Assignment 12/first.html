Serial No String method or Property Example Code Description
1 concat() let text1 = "sea";
let text2 = "food";
let result = text1.concat(text2);		 The concat() method joins two or more strings.The concat() method does not change the existing strings.The concat() method returns a new string.
2 length let text = "Hello World!";
let length = text.length;
 The length property returns the length of a string.The length property of an empty string is 0.
3 toLowerCase() let text = "Hello World!";
let result = text.toLowerCase();
 The length property returns the length of a string. The length property of an empty string is 0.The toLowerCase() method converts a string to lowercase letters.The toLowerCase() method does not change the original string. Se also toLocaleLowerCase() and toUpperCase()
4 toUpperCase() let text = "Hello World!";
let result = text.toUpperCase();
 The toUpperCase() method converts a string to uppercase letters.The toUpperCase() method does not change the original string. Se also toLocaleUpperCase() and toLowerCase()
5 indexOf() let text = "Hello world, welcome to the universe.";
let result = text.indexOf("welcome");
 The indexOf() method returns the position of the first occurrence of a specified value in a string. The indexOf() method returns -1 if the value is not found. The indexOf() method is case sensitive. See also the lastIndexOf() method.
6 lastIndexOf() let text = "Hello planet earth, you are a great planet.";
let result = text.lastIndexOf("planet");
The lastIndexOf() method returns the index from the beginning (position 0). The lastIndexOf() method returns -1 if the value is not found. The lastIndexOf() method is case sensitive.
7 charAt() let text = "HELLO WORLD";
let letter = text.charAt(0);
The charAt() method returns the character at a specified index (position) in a string. The index of the first character is 0, the second 1, ...The index of the last character is string length - 1 (See Examples below)See also the charCodeAt() method.
8 endsWith() let text = "HELLO WORLD";
let result = text.endsWith("world");
 The endsWith() method returns true if a string ends with a specified string. Otherwise it returns false. The endsWith() method is case sensitive.
9 includes() let text = "Hello world, welcome to the universe.";
let result = text.includes("world");
 The includes() method returns true if a string contains a specified string. Otherwise it returns false.The includes() method is case sensitive.
10 substring() let text = "Hello world!";
let result = text.substring(1, 4);		 The substring() method extracts characters, between to indices (positions), from a string, and returns the substring. The substring() method extracts characters from start to end (excusive). The substring() method does not change the original string.
11 slice() const fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
const citrus = fruits.slice(1, 3);		 slice() returns selected elements in an array, as a new array. slice() selects from a given start, up to a (not inclusive) given end. slice() does not change the original array.
12 split() let text = "How are you doing today?";
const myArray = text.split(" ");		 The split() method splits a string into an array of substrings. The split() method returns the new array. The split() method does not change the original string. If (" ") is used as separator, the string is split between words.
13 replace() let text = "Visit Microsoft!";
let result = text.replace("Microsoft", "W3Schools"); 		The repeat() method returns string with a number of copies of a string. The repeat() method returns a new string.The repeat() method does not change the string it copies from.
