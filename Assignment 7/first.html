ARRAY METHOD

  1. Please write all the names of the array methods which you have learned so far. With a single line of description about each method.

  • pop() -> The pop() method removes the last element from an array

  • push() -> The push() method adds a new element to an array(at the end)

  • shift() -> The shift() method removes the first array element and "shifts" all other elements to a lower index.

  • unshift() -> The unshift() method adds a new element to an array (at the begining),and "unshifts" older elements.

  • splice() -> The splice() method can be used to add new item to an array and to remove elements without leaving "holes" in the array

  • slice() -> The slice() method slice out a piece of an array into a new array

  • concat() -> The concat() method creates a new array by merging (concatenating) existing arrays

  • reverse() -> Reverse the order of the element in place.(First becomes the last, last becomes first.)

  • toString() -> The javascript method toString() converts an array to a strinf of (comma seperated) array values.

  • forEach() -> forEach() calls a function for each elements in an array. forEach() is not executed for empty elements.
